## 4. 虚拟环境管理
- 入口：侧栏 “Ruyi” 选项卡
- 覆盖点：
    - 自动检测并列出现有虚拟环境
    - 图形化创建虚拟环境
        - 选择 toolchain（列表仅显示已安装项）
        - 未安装的 toolchain 触发自动安装提示
        - 可选第二个及以上 toolchain（按 Esc 跳过）
        - 是否需要模拟器、环境名称、环境路径、sysroot 路径等参数填写
    - 创建完成后在侧栏管理与激活
- **限制说明：目前暂不支持复杂 profile**
- 期望结果：创建流程顺畅，参数校验清晰，环境可正常激活与切换
## 5. 解压项目模板与编译
- 入口：Explorer 面板文件树右键
- 覆盖点：
    - 通过 Ruyi 包管理器解压已有项目模板
    - 结合已创建的虚拟环境与所选 toolchain 进行编译
- 期望结果：
    - 模板解压成功，目录结构正确
    - 选择虚拟环境后编译过程可完成并生成产物
