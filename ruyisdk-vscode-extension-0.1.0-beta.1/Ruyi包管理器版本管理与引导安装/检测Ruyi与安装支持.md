# Ruyi检测与安装支持功能测试

## 操作步骤

1. 确认未安装RuyiSDK且VScode已安装插件
2. 打开VScode

## 预期结果

启动自动检测，缺失RuyiSDK时右下角弹出提示窗，提示安装或打开指南

## 测试结果

一键安装失败，在点击install按钮后出现警告，随后弹出下载失败窗口，并提示打开指南
